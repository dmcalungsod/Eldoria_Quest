## 📯 EventHerald

**Date:** 2026-03-03
**Status:** Active

**Summary:**
Designed and implemented the "Frostfall Expedition" event to encourage exploration of the new Frostfall Expanse region.

**Deliverables:**

### ❄️ Frostfall Expedition Event
*   **New Event Type:** `FROSTFALL_EXPEDITION` added to `WorldEventSystem`.
*   **Mechanics:**
    *   **Reduced Threat:** 10% damage reduction from monsters in Frostfall Expanse (encourages lower ranks to try).
    *   **Increased Loot:** 50% increase in drop rates specifically for Frostfall items (`frost_crystal`, `winter_wolf_pelt`, etc.).
    *   **Global Boosts:** 25% global loot boost, 10% global EXP boost.

**Files Modified:**
*   `game_systems/events/world_event_system.py`: Added event config.
*   `game_systems/adventure/adventure_session.py`: Updated logic to apply threat reduction in Frostfall.
*   `game_systems/adventure/adventure_rewards.py`: Updated logic to apply loot bonus in Frostfall.
*   `tests/test_frostfall_event.py`: Added verification tests.

**Next Steps:**
*   @StoryWeaver can draft an announcement for the event start.
*   @GameForge can verify if drop rates feel rewarding enough during playtesting.
