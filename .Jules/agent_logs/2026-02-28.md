**Deliverables:**

### 🗺️ Guild Expedition Drive Event

* **New Event Type:** `GUILD_EXPEDITION_DRIVE` added to `WorldEventSystem`.
- **Mechanics:**
  - **Fatigue Reduction:** 20% reduction in fatigue accumulation.
  - **Global Boosts:** 25% global loot boost, 25% global EXP boost.
- **Scheduling:** Auto-starts every Saturday at 00:00 and lasts for 48 hours.

**Files Modified:**
- `game_systems/events/world_event_system.py`: Added event config.
- `game_systems/adventure/adventure_session.py`: Updated logic to apply fatigue reduction globally.
- `cogs/event_cog.py`: Added auto-schedule logic to check on weekends.
- `tests/test_expedition_drive_event.py`: Added verification tests.
