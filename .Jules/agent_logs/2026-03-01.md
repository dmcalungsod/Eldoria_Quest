### ⚡ Batch Context Bundle Retrieval
- **Problem:** `AdventureResolutionEngine.resolve_sessions_batch` iterated over each completing session and called `DatabaseManager.get_combat_context_bundle` individually. If 1,000 players completed adventures at once, this generated 1,000 queries.
- **Solution:** Added `get_combat_context_bundles_batch(discord_ids)` to `DatabaseManager`. It leverages the MongoDB `$in` operator inside an aggregation pipeline to fetch all context bundles (stats, buffs, skills, session) for all resolving players in a single DB round-trip.
- **Impact:** Decreased the database query count in `AdventureResolutionEngine.resolve_sessions_batch` from O(N) to O(1).
