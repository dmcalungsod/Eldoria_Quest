# Security Policy

## Supported Versions

Currently we only provide security updates for the `main` branch and the latest release tag.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| others  | :x:                |

## Reporting a Vulnerability

We take the security of Eldoria Quest very seriously. If you have discovered a security vulnerability in this project, we appreciate your help in disclosing it to us in a responsible manner.

Please do **NOT** report security vulnerabilities through public GitHub issues.

Instead, please report them to the repository maintainer directly by sending a private message or via the private vulnerability reporting tool built into GitHub.

When reporting a vulnerability, please include:

- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact on users or the system.

We will acknowledge receipt of your vulnerability report as soon as possible, and we will send you regular updates about our progress in addressing the issue.
